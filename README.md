# Pizza-Web-Application
Pizza ordering website using raw HTML, CSS, Javascript.

Build a webpage that displays a selection of Pizza, Side order, Drinks and Desserts.  
There must be at least 15 pizzas to offer, and one pizza option should be a 50/50 option, where the user can select the size of their pizza and which pizza to select for either side of their order.  
Pizzas will be of 4 varying sizes: small, medium, large, extra-large.  
The client wants a theme for this pizza page and have opted for movies, as they are located near a cinema.  
Use your imagination to build this page and allow the user to select their order and place it in a shopping cart.  There should be a running total for order.  
Ensure that your page has images and consider the user experience when building this.

Finally, using Selenium (Selenium) test your application - This autonomous tool is used throughout the industry to perform tests on web applications.
